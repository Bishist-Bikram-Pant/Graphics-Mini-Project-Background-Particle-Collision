# Particle Collision Animation

An interactive canvas-based particle system with collision detection and dynamic connections.

## Features

- Particle collision physics
- Interactive controls for particle count and size
- Dynamic particle connections based on distance
- Responsive canvas

## Usage

Open `index.html` in a web browser.

## Controls

- Adjust particle count (10-200)
- Adjust particle size (1-10)
- Reset button to restore defaults
